基本元件實作 - 使用 Google Fonts

不管是網頁還是 App，甚至是文件、簡報、試算表...等，都會遭遇到同樣的一個議題：字體，雖然我們在自己電腦上可能安裝了千百種字體格式，一旦要發佈到網路上，或是要包裝成一個安裝檔案，都會變得相當的複雜且麻煩。
幸好隨著時代的進步，像是 Google 就提供大量的免費字體讓使用者直接連結，只要在 CSS 內指定特殊字體，不管使用者在世界上任何一個角落，只要有網路，就能夠看到同樣的字體，是不是相當的便利呢！EZoApp 因為由 CSS 的樣式表來建構樣式，所以同樣支援開放字體，這個章節主要就是要介紹，如何使用 Google Fonts。( 不過比較可惜，目前都還只有英文字體 )

範例連結：
基本元件實作 - 使用 Google Fonts

這邊也提供幾個不錯的公開字體網站給大家使用：
Google Fonts ( 目前最夯的網路公開字體來源 )
Adobe® Edge Web Fonts ( Adobe 出品的網路公開字體 )
Open Font Library ( 國外的網路公開字體來源 )


一、載入 Google Fonts

首先我們必須連上 Google Fonts 的網站，藉由他所提供的字體連結，將字體載入到我們的 CSS 當中，連上 Google Fonts 之後，可以看到 Google 提供的一大堆免費字體，找到想到的字體後，點選 quick use，就可以看到相關的操作連結和說明。


點選後往下拉，可以看到「3. Add this code to your website:」，我們選擇 @import，就是要將此字體載入 CSS 的程式碼。


複製這串程式碼，回到 EZoApp 的 CSS 編輯畫面，將它貼在最上方，接著我們就可以使用這組字體囉！( 這邊有個要注意的地方，沒有網路的情況下，就會自動使用預設字體 )


二、使用 Google Fonts

打開 EZoApp 的編輯畫面，貼上剛剛複製的程式碼，然後在我們需要用到這組字體的樣式上加上字體名稱，就可以立馬在編輯區域中看到字體已經套用了 Google Fonts，是不是非常的方便實用呢！


當然大家也可以不用一定要選擇 Google Fonts，也可以使用自己的字體，不過自己的自己缺點就是必須放在自己的空間，而且也可能會有相關的版權問題，因此免費又有授權的字體，當然就選擇這些大廠的公開字體囉！( 不過中文字就算是公開字體，還是鮮少有人使用，畢竟一套中文字體可以多達 5MB 以上，使用者大概不會有耐心等待這麼大的檔案下載吧 )
使用自己的字體的方式： 

@font-face {  
  font-family: "font-name";  
  src: url("http://xxx.com/abc.ttf"); 
 }
