EZoApp

結合 javascript 應用 - 串接公開資料

大家一定有使用過一些介紹各地美食或景點的網站或 app，甚至最近最夯的腳踏車資訊、公車資訊...等交通生活資訊也都可以藉由各種平台查詢，因此內心不免好奇這些資訊到底是從哪裡來的？其實許多的資訊，在各個政府機關的網站上，都有提供相關的公開資料讓民眾連結查詢，EZoApp 同樣提供了這個功能，藉由連結政府機關公開資料的 json 檔案，就能夠輕鬆獲取數據，製作相關的 app。
不過這裡有個小問題要提醒大家的，因為我們是在電腦上操作，而電腦的瀏覽器對於獲取外部網站的資料有著比較嚴格的把關機制，因此我們直接使用瀏覽器開啟會有問題，如果要測試，我們必須使用 chrome 瀏覽器，並且把某個安全機制關閉，就能夠進行測試，下列會教導大家如何使用。
