結合 JavaScript 應用 - 串接開放資料

大家一定有使用過一些介紹各地美食或景點的網站或 app，甚至最近最夯的腳踏車資訊、公車資訊...等交通生活資訊也都可以藉由各種平台查詢，因此內心不免好奇這些資訊到底是從哪裡來的？其實許多的資訊，在各個政府機關的網站上，都有提供相關的公開資料讓民眾連結查詢，EZoApp 同樣提供了這個功能，藉由連結政府機關公開資料的 JSON 檔案，就能夠輕鬆獲取數據，製作相關的 app。
不過這裡有個小問題要提醒大家的，因為我們是在電腦上操作，而電腦的瀏覽器對於獲取外部網站的資料有著比較嚴格的把關機制，因此我們直接使用瀏覽器開啟會有問題，如果要測試，我們必須使用 Chrome 瀏覽器，並且把某個安全機制關閉，就能夠進行測試，下列會教導大家如何使用。

結合 JavaScript 應用 - 串接開放資料

一、找一個開放資料平台獲取檔案

想當然爾，要串接開放資料一定得有開放資料的檔案，這裡我們以高雄市政府的開放資料平台作為例子，當然大家也可以去其他開放資料獲取資料，進入開放資料平台之後，我們選擇「高雄旅遊網-餐飲資料」作為我們的資料來源。
連結： 高雄市政府的開放資料平台


二、將瀏覽器的安全性關閉

如果我們要在電腦上的瀏覽器測試，就必須先把瀏覽器的某項安全性關閉，這是電腦的瀏覽器預設的規則，不過如果之後發佈到行動裝置上，就不會有這些安全性開關的問題，也因此如果要測試，就要先下指令關閉瀏覽器的設定。
設定的方法就是在 Chrome 的圖示上按右鍵 ( 這裡我們以 Chrome 作為範例 )，選擇「內容」，在「捷徑」的頁籤內找到「目標」的欄位，在 Chrome.exe 檔名後方加入以下文字： --args --disable-web-security，將 Chrome 的安全性關閉，就可以開始進行測試了


三、新增頁面

在範例中我們做了四個頁面，第一頁是入口頁，指向不同的資料 ( 餐飲資訊與玩樂資訊 )，第二頁為串接資料後呈現的店家與景點清單，第三頁是點選清單後的詳細訊息，第四頁是地圖。

五、撰寫 JavaScript

接著就是撰寫 JavaScript 檔案，這個範例使用比較簡易的方式，只是將各個資訊寫入標籤內。 

var mapAddress;
$(document).on('pageinit', function () {
  $('#home_btn a').on('click', function (e) {
    var headerTitle = $(this).find('h4').text();
    $('#title').text(headerTitle);
  });

  $('#a1').on('click', function (e) {
    listContent("http://data.kaohsiung.gov.tw/Opendata/DownLoad.aspx?Type=2&CaseNo1=AV&CaseNo2=2&FileType=1&Lang=C");
  });

  $('#a2').on('click', function (e) {
    listContent("http://data.kaohsiung.gov.tw/Opendata/DownLoad.aspx?Type=2&CaseNo1=AV&CaseNo2=1&FileType=1&Lang=C");
  }); 

function listContent(service) {
    var url = service;
    if (url) {
      $.getJSON(url).complete(function (data) {
        $("#gk-219CcdN").gk("model", $.parseJSON(data.responseText));

        $('#gk-219CcdN a').on('click', function (e) {
          var g1 = $(this).attr('s1');
          var g2 = $(this).attr('s2');
          var g3 = $(this).attr('s3');
          var g4 = $(this).attr('s4');
          var g5 = $(this).attr('s5');
          $('.d1').text(g1);
          $('.d2').text(g2);
          $('.d3').text(g3);
          $('.d4').text(g4);
          $('.d5').text(g5);
          mapAddress = g2;
        });
      });
    }
  }
});
$(document).on('pageshow', '#MapPage', function () {
  $('#map1').gk('address', mapAddress);
});

程式都寫好之後，就可以串接開放資料，也實際做出一個簡易的查詢平台了！
