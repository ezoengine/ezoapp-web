Integrating JavaScript in your web - Touch Sliding

This paragraph will talk about a rarely know but useful function: touch access. It’s usually a big problem for touch access when using the mobile web or creating hybrid app. A simple distinction between behaviors, such as tap, mousedown, mouseover and so on, will make programming being complicated. Fortunately, jQuery Mobile helps us in solving these problems.
The existed 5 touch events, inclusive of the basic gesture behaviors, can make a 3-dimensional transition by using transform 3D tech of CSS3 together. However, the way of multi-touched is not included.

Usage is very simple, just like jQuery:. On ('tap', function () {}) or on ('swipe', function () {}). General case button click event bound almost exactly the same, and because of this, You can spend time on the details of the treatment effects or content.

     <div id="box3" class="box-child">3</div>

Then we just have to let the package with four div(s) box rotated 90 degrees each time when sliding or clicking the button and the three-dimensional transition effects will be showed. Remember to set the attribute as “presere-3d” for the transform-type, or the 4 div(s) won’t be in tree-dimensional because of the default attribute value is ‘flat’.

Then see javasctript, In the example, the first four buttons at the bottom to set "on. ('Click', function () {})" event, Slide each time, Will trigger the corresponding button, Why do it?
Because we want to slide when Button also corresponds to color, To make the program even more simple, Only the switching behavior tied to the button Just do not need a judge, The following program is the first of a button javascript, angle is the angle the outset to declare variables The main box is to know how much of an angle. 

   $('#b1 a').on('click', function () {
     $('#box').css({
       '-webkit-transform': 'rotateY(0deg)',
       '-moz-transform': 'rotateY(0deg)',
       'transform': 'rotateY(0deg)'
     });
     angle = 0;
   });

It’s a simple example of JS about swiping to the left by using angle judgment. When sliding to 1 and it won’t move anymore.

    $('#home').on('swipeleft', function () {
     if (angle <= 0 && angle > -270) {
       angle = angle - 90;
       switch (angle) {
       case -90:
         $('#b2 a').click();
         break;
       case -180:
         $('#b3 a').click();
         break;
       case -270:
         $('#b4 a').click();
         break;
       }
     }
   });

On this way by sliding the trigger button, you can make switching operation precisely, with the CSS3 3D transform effect, you can also present quite a three-dimensional transition effects.
In order to make the page with more sheets by using rotation, you can calculate the angle of rotation dynamically, which quite convenient to be used as a desktop or albums.
