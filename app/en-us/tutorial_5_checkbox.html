Basic component hands-on practice – CheckBox design

In addition to the pull-down menu introduced in the previous section, checkbox is another frequently seen feature in web forms and app forms. In the past, using CSS, these types of form elements were difficult to modify; and they even appeared differently in different browsers. It’s quite a headache.
But, now, with EZoApp, you can quickly create checkboxes using the EZoApp’s drag-and-drop method and arrange them horizontally and vertically by using mouse clicks only. These checkboxes also use jQuery Mobile’ standard styles, styles that are frequently seen on mobile devices, thus avoiding lots of unnecessary troublesome work.
If you are not satisfied with the basic black and white styles, you can try to edit them with HTML or CSS. You can refer to the previous sections of "Creating customized buttons" and "Using Google fonts"; there are some good tutorials in them.

I. Adding a checkbox component

As usual, using EZoApp is about using its handy features. Use the drag-and-drop method to drag a controlgroup-checkbox component into the editing area directly, a Checkbox component is created automatically.


II. Adding options

Next, use the method introduced earlier—click the options of checkbox, and then the green plus sign on the upper-right corner, new options are added automatically.


III. Changing vertical and horizontal directions

What do we do if we want to change the vertical or horizontal direction? Just open the attribute panel and change the direction!


IV. Writing CSS to modify styles

It has been pretty straight forward up to this point. Basically, everything can be done by using just the mouse. In this step, we will modify the styles directly to make the checkbox more appealing. The way to change it is by modifying the style of the legend tag of the title. You can paste in the red program code directly to see the change of the style. 

<legend style="-webkit-border-radius:10px 10px 0 0; background:-webkit-linear-gradient(#99f,#33a); text-shadow:none; color:#fff; width:279px; height:30px; padding-top:7px; padding-left:10px; margin-bottom:-6px;">Checkbox Group</legend>
