Basic component hands-on practice – Creating droplist menus

Droplist menus are not just common in web pages, they also appear in apps from time to time. Programs linking to back-end databases also often use droplist menus to present their many functions and options.
In EZoApp, droplist menus are constructed using jQuery Mobile components as the foundation. Users can simply use the drag-and-drop method to design droplist menus without writing extra code. This also makes it more convenient for the back-end staff to work with the front-end staff, thus allowing developers to save time spent in writing code for droplist menus and focus their efforts on functions and details that deserve more attention.
As expected, droplist menus in EZoApp are all created in HTML. Therefore, if you have a certain level of understanding in HTML, you will be able to add more modifications and variations to the droplist menus and present a more wonderful screen! For those who are not familiar with HTML, they can also learn how droplist menus are presented in HTML through the two-way interaction in EZoApp. This might as well be a good way to learn!

I. Adding a droplist menu

To add a droplist menu is easy. You only need to drag the selectmenu component into the editing area.


II. Adding options for the droplist menu

Because this is a droplist menu, we need to add the options in it manually. Adding the options is almost as easy as copy and paste. We just need to copy the option tags in the program code. The number of option tags determines the number of options. Lastly, do not forget to change the value and the titles of the options!



III. Assembling ListView

You might have already discovered that we can separate the elements of the droplist menu component in the editing area. For example, we can drag the droplist menu outside of the dashed box, or drag the title outside of the dashed box and so on. We can even assemble different elements together by dragging. Here, however, we found that we couldn't assemble the selectmenu of the droplist menu and ListView together. Don’t worry about this. Because all the programs and components basically consist of HTML syntax, all what we need to do is the simple copy-and-paste action in order to easily assemble different components!


With this method, we can freely assemble components. However, there’s one thing to keep in mind—try your best to follow the specifications of the HTML syntax in order to avoid unnecessary issues and mistakes later.
