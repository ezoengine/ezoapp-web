一、放上按钮

我们直接使用拖拉的方式，把左侧的 button组件拖曳进入编辑画面中，就会自动产生一个按钮。


二、快速复制按钮

当鼠标点击编辑区域中的按钮，会发现元见右上角出现了几个小 icon，几个小 icon 的意思分别为：「绿色加号」复制并新增该组件，「红色叉叉」删除该组件，「黑色向上箭头」选取父层组件，如果我们要快速复制按钮，只要点选绿色加号，按钮就会自动产生。


三、认识按钮基本样式

点选按钮后，我们便可以使用属性面板来调整按钮的属性，调整属性后，相关属性的样式名称也会加入程序代码内，，这里我们可以设定按钮 icon 的位置、按钮的大小...等属性，而其他比较进阶的属性，后续就利用 CSS 来调整，下方列出范例档案中13个按钮的基本样式。

四、建立按钮样式表单

按钮组件放上编辑区域后，最后我们就是要利用 CSS 来编辑按钮的样式，除了可以控制长宽之外，也可以利用以下几个设定来设定按钮的样式：

  1. -webkit-box-radius:后方输入数值，来设定按钮四个角为圆角 (适用于 Chrome，Firefox 改为 -moz-box-radius，IE 改为 box-radius)，例如：

    -webkit-border-radius: 20px; //四个圆角的半径为20px

  2. -webkit-box-shadow:后方输入数值，来设定按钮阴影 (适用于 Chrome，Firefox 改为 -moz-box-shadow，IE 改为 box-shadow)，例如：

    -webkit-box-shadow: inset #ddd 0 0 10px; //内阴影
    -webkit-box-shadow: #ddd 0 0 10px;       //外阴影

  3. background:后方可带入背景图片，但因为 jQuery Mobile 组件的默认值，必须加入 !important 才能顺利呈现，例如：

    background: url(图片网址) center center no-repeat!important;

  4. display:因为按钮本身是区块 ( block ) 属性，如果要让按钮可以一个接一个的排下去，最简单的方法就是调整 display，例如：

    display: inline-block;

上面列出几个比较特别需要注意的 CSS 样式，其他的其实都是基本的 CSS，大家可以自行撰写，并且立即看到效果，是不是相当的方便迅速呢！
