基本组件实作 - 使用 Google Fonts

不管是网页还是 App，甚至是文件、简报、电子表格...等，都会遭遇到同样的一个议题：字体，虽然我们在自己计算机上可能安装了千百种字体格式，一旦要发布到网络上，或是要包装成一个安装档案，都会变得相当的复杂且麻烦。
幸好随着时代的进步，像是 Google 就提供大量的免费字体让用户直接链接，只要在 CSS 内指定特殊字体，不管用户在世界上任何一个角落，只要有网络，就能够看到同样的字体，是不是相当的便利呢！EZoApp 因为由 CSS 的样式表单来建构样式，所以同样支持开放字体，这个章节主要就是要介绍，如何使用 Google Fonts。( 不过比较可惜，目前都还只有英文字体 )

这边也提供几个不错的公开字体网站给大家使用：
Google Fonts ( 目前最夯的网络公开字体来源 )
Adobe® Edge Web Fonts ( Adobe 出品的网络公开字体 )
Open Font Library ( 国外的网络公开字体来源 )


一、载入 Google Fonts

首先我们必须连上 Google Fonts 的网站，藉由他所提供的字体链接，将字体加载到我们的 CSS 当中，连上 Google Fonts 之后，可以看到 Google 提供的一大堆免费字体，找到想到的字体后，点选 quick use，就可以看到相关的操作连结和说明。


点选后往下拉，可以看到「3. Add this code to your website:」，我们选择 @import，就是要将此字体加载 CSS 的程序代码。


复制这串程序代码，回到 EZoApp 的 CSS 编辑画面，将它贴在最上方，接着我们就可以使用这组字体啰！( 这边有个要注意的地方，没有网络的情况下，就会自动使用默认字体 )


二、使用 Google Fonts

打开 EZoApp 的编辑画面，贴上刚刚复制的程序代码，然后在我们需要用到这组字体的样式上加上字体名称，就可以立马在编辑区域中看到字体已经套用了 Google Fonts，是不是非常的方便实用呢！


当然大家也可以不用一定要选择 Google Fonts，也可以使用自己的字体，不过自己的自己缺点就是必须放在自己的空间，而且也可能会有相关的版权问题，因此免费又有授权的字体，当然就选择这些大厂的公开字体啰！( 不过中文字就算是公开字体，还是鲜少有人使用，毕竟一套中文字体可以多达 5MB 以上，使用者大概不会有耐心等待这么大的档案下载吧 )
使用自己的字体的方式： 

@font-face {  
  font-family: "font-name";  
  src: url("http://xxx.com/abc.ttf"); 
 }
